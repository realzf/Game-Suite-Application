# Project Title

Tic-Tac-Toe

## Description
This project is Tic-Tac-Toe. There are two game mode, classical tictactoe and numerical tictactoe. It is played on a 3 by 3 grid. There are two players. The first player to get 3 of their symbols/numbers adding up to 15 on the board in a row is the winner. A tie happens when the board is full and no one got 3 in a row/numbers adding up to 15.

### Executing program

Compiling/running the program: The program should be compiled using Gradle

1. Type first command then press enter.
```
gradle build
```
2. Second command
```
gradle run
```
3. Third command to run console version
```
java -cp build/classes/java/main tictactoe.TextUI
```
4. To run GUI version

```
java -jar build/libs/A3.jar
```
